---
title: German TV Mediathek
---

If you are native German speaker and want to consume public TV station
content using Liferea you can do so by following the RSS feeds provided
by <a href="https://mediathekviewweb.de/">https://mediathekviewweb.de/</a>.

To easily import some of the most important feeds here is an <a href="/liferea/blog/assets/german-tv.opml">OPML file</a>
that you can download and import in Liferea using `Subscriptions` -> `Import Feedlist`.
