Additional Social Bookmarking Sites

The default social bookmarking sites delivered with Liferea might not be enough for you. Using the scripting support in Liferea you can redefine the social bookmarking site.

To do so open up the script manager from the "Program" menu and select the startup hook. Create a new script and insert something like given in the following examples:

-----------------------------------------------------------
-- Registers an additional social bookmarking site
-----------------------------------------------------------

-- Example with URL posting only
liferea.social_register_site("MySite","http://mysite.org?%s",0,0)

-- Example with URL and title posting (title passed at second position)
liferea.social_register_site("MySite2","http://mysite.org?url=%s&title=%s",1,0)

-- Example with URL and title posting (title passed at first position)
liferea.social_register_site("MySite3","http://mysite.org?title=%s&url=%s",1,1)

The URL format is your choice as long as it contains the correct number of %s occurences. The exact meaning of the parameters should be intuitive but can be found in social.h.

When using Liferea 1.5.x: Please ensure to use liferea.social_register_bookmark_site() instead of liferea.social_site(). With the 1.5 series there is also a new function liferea.social_register_link_search_site() to register additional search engines to be used with the "link cosmos" link in the item view.
